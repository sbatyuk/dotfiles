## Misc

| Origin        | Maps to             | Comment                           |
| ------------- | ------------------- | --------------------------------- |
| `✱` + /       | `⇧⌘⌃4`              | Screenshot of area to clipboard   |
| `✱` + `⌘` + / | `⇧⌘4`               | Screenshot of area to file        |
| `✱` + X       | `⌃␢`  Ctrl+Spacebar | Switch Input Source               |
| `esc`         | `⌃⌘Q`               | Lock screen                       |
| `✱` + `Y`     | `⌃⌘␢`               | Native macOS emoji keyboard popup |
